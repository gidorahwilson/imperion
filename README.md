# imperion
🎮 Imperion — Web Game Distribution Platform

Imperion adalah web aplikasi berbasis Ruby Sinatra yang berfungsi sebagai portal distribusi dan katalog game dengan tampilan modern, ringan, dan terstruktur.

Project ini menampilkan bagaimana membangun UI portal profesional dan sistem navigasi konten menggunakan arsitektur server-side sederhana tanpa framework frontend besar.

✨ Konsep Aplikasi

Imperion bukan game, melainkan:

Web App Game Hub tempat pengguna dapat melihat, menelusuri, dan mengunduh game dengan tampilan modern seperti platform distribusi game.

Fokus utama project ini adalah:

Desain UI/UX profesional
Struktur layout enterprise
Navigasi konten yang rapi
Implementasi server-side rendering dengan ERB
🚀 Fitur Utama
Navbar profesional & responsif
Hero section modern
Grid katalog game (card system)
Halaman Top Downloads, About, Contact
Desain dark theme konsisten
Server-side rendering dengan ERB
Tanpa JavaScript kompleks
